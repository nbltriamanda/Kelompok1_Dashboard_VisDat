### ğŸ” **Variabel dalam Dataset**

|       Nama Variabel      |                       Tipe Data                              |                       Keterangan                          |
|:-------------------------|:-------------------------------------------------------------|:----------------------------------------------------------|
| Jenis Kelamin            | Kategorik (laki-laki, perempuan, non-biner)                  | Identitas gender individu                                 |
| Penghasilan              | Numerik                                                      | Pendapatan tahunan individu.                              |
